Caracteres especiais

Para imprimir na tela os caracteres especiais, é preciso inserir o código da entidade encontrada clicando aqui

Caracteres especiais são aqueles que são usados dentro do HTML, como o <, o > ou o /, por exemplo

Exemplo: <p> - Dessa forma, é possível imprimir uma tag que geralmente não é impressa mas que está presente no código HTML

Outro auxílio com caracteres especiais é colocar espaçamento.
Dentro do código, se colocar vários espaços em branco, estes não serão impressos
